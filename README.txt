Техническая документация.

Проект состоит из пяти скриптов:
1. GameController отвечает за:
    - появление в рондомных местах на карте игровых объектов(Heroes и Zombie), 
    при загрузке или перезапуске уровня.
    - метод Restart - перезапуск уровня.

2. Enemy Controller отвечает за управление объектом  Zombie  и его уничтожение.

3. Gun за управление стрельбой

4. Player  отвечает за управление объектом Heroes.

5.PlayerMain - Scriptableobject, где хранятся настройки скорости пули и скорости объекта Heroes.

Пользовательская документация игры.

Суть игры - добраться до башни с которой можно убить Zombie.
- Управление персонажем осуществляется левой кнопкой мыши, нужно кликнуть на место в которое необходимо попасть.
- Режим стрельбы возможен только когда объект Heroes оказывается на башни.
- Игра оканчивается после убийства объекта Zombie.
